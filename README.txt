Калькулятор проведённых уроков

Как запустить:
1. Распакуйте архив.
2. Откройте файл index.html двойным кликом в браузере.

Что умеет программа:
- считает количество занятий между датой начала и датой окончания;
- учитывает выбранные дни недели;
- автоматически исключает официальные праздничные дни РФ за 2023–2036 годы;
- позволяет вручную добавлять дополнительные праздники, каникулы и переносы через кнопку «Добавить праздник»;
- показывает список автоматических праздников в выбранном периоде.

Важно:
В calendar.js заложены официальные праздничные дни по ст. 112 ТК РФ.
Для будущих лет производственные переносы могут быть утверждены позднее, поэтому их можно добавлять вручную через интерфейс программы.
